## Goal & hard constraints

- Decode FT8 (and FT4) from receiver audio — **live** (slot-aligned capture) and
  from a **recorded WAV** — and present the (many, simultaneous) decodes usefully
  in a REPL-only setting. User wants to try it live or from a recording.
- **Minimal dependencies** for the decoder: no C, no FFI. The sync/demod/LDPC/
  unpack chain is all our own Rust; the forward FFT is the `realfft`/`rustfft`
  crate (it replaced our hand-rolled FFT, ~10x faster). Other deps are just
  `serde`/`tracing` (serialization + logging), nothing else algorithmic.
- The algorithms are reverse-engineered / ported from **ft8_lib** (MIT, Kārlis
  Goba). The fixed numeric tables are the **WSJT-X** (Franke/Taylor) on-air spec.
  Attribution obligations are real and documented in ATTRIBUTION.md +
  THIRD_PARTY_LICENSES/. Keep them.

## Key technical decisions (don't re-derive)

- **Sample rate = 12000 Hz** internally (reuse `audio::dsp::Resampler`, built
  48k→12k). FT8: symbol 0.16 s → `block_size=1920`; `time_osr=2`, `freq_osr=2` →
  **nfft=3840**, bin spacing 3.125 Hz (= 6.25 Hz tone / freq_osr). FT4: symbol
  0.048 s → block_size=576, nfft=1152.
- **Waterfall mag storage**: like ft8_lib, store each bin as `u8` = clamp(`2*db +
  240`, 0..255) where `db = 10*log10(1e-12 + |X|²)`. `WF_ELEM_MAG(x) = x*0.5 -
  120`. Window = Hann × fft_norm where `fft_norm = 2/nfft`. min_bin =
  `f_min*symbol_period` (use f_min=200), max_bin = `f_max*symbol_period`+1
  (f_max=3000).
- **Sync search ranges** (find_candidates): time_sub 0..time_osr, freq_sub
  0..freq_osr, **time_offset −10..20**, freq_offset 0..(num_bins−num_tones+1).
  min_score=10, keep top **140** candidates (min-heap), LDPC 25 iters.
- **normalize_logl**: scale 174 LLRs by `sqrt(24 / variance)` before bp_decode.
- **Callsign hash**: `n58 = base38(callsign, AlphanumSpaceSlash, up to 11 chars,
  space-padded)`; `n22 = (47055833459u64.wrapping_mul(n58) >> 42) & 0x3FFFFF`
  (the multiply **overflows — must use wrapping_mul**). n12 = n22>>10, n10 =
  n22>>12. Keep a session-lived `CallHash { HashMap<u32 n22, String> }`; lookup by
  shift (22→0, 12→10, 10→12).

## NEXT: `src/message.rs` (detailed recipe)

Constants: `MAX22=4_194_304`, `NTOKENS=2_063_592`, `MAXGRID4=32_400`.
Payload = `[u8;10]` (77 bits). Public `MessageType` enum (FreeText, Standard,
NonStd, Telemetry, …). Carry a `CallHash` (see above).

`get_i3 = (p[9]>>3)&7`. `get_n3 = ((p[8]<<2)&4)|((p[9]>>6)&3)`. `get_type`: i3==0
→ n3 switch (0 FreeText,1 DXped,2 EU_VHF,3/4 ARRL_FD,5 Telemetry); i3 1|2
Standard; 3 ARRL_RTTY; 4 NonStd; 5 WWROF.

**Decode (need these for normal traffic): Standard (type 1/2), FreeText, NonStd
(type 4), Telemetry-hex.** Bit extraction (cast each `p[i]` to u32 BEFORE
shifting):

- decode_std: `n29a=(p0<<21)|(p1<<13)|(p2<<5)|(p3>>3)`;
  `n29b=((p3&7)<<26)|(p4<<18)|(p5<<10)|(p6<<2)|(p7>>6)`; `ir=(p7&0x20)>>5`;
  `igrid4=((p7&0x1F)<<10)|(p8<<2)|(p9>>6)`. Then `unpack28(n29a>>1, n29a&1, i3)`
  → call_to, same for n29b → call_de, `unpackgrid(igrid4, ir)` → extra.
- `unpack28(n28, ip, i3)`: if n28<NTOKENS → tokens (0 DE,1 QRZ,2 CQ; ≤1002 "CQ
  "+int_to_dd(n28−3,3,false); ≤532443 "CQ "+4×base27 LettersSpace of (n28−1003)).
  Else n28−=NTOKENS; if <MAX22 → lookup_callsign(22,n28). Else n=n28−MAX22; build
  6-char call: `[5,4,3]=charn(n%27,LettersSpace)` (÷27 each), `[2]=charn(n%10,
  Numeric)` ÷10, `[1]=charn(n%36,Alphanum)` ÷36, `[0]=charn(n%37,AlphanumSpace)`.
  Swaziland (3D0→3DA0) / Guinea (Qx→3X) fix-ups; trim; len<3 → fail; if ip: i3==1
  append "/R", i3==2 "/P". save_callsign(result).
- `unpackgrid(igrid4, ir)`: if ≤MAXGRID4 → 4-char grid `[0,1]='A'+ (÷18),
  [2,3]='0'+ (÷10)`, prefix "R " if ir>0. Else irpt=igrid4−MAXGRID4: 1→"" ,2 RRR,
  3 RR73,4 73, default → (ir>0 ? "R":"") + int_to_dd(irpt−35, 2, true).
- decode_nonstd (port message.c:504): n12, n58 (see below), iflip, nrpt, icq;
  unpack58(n58)→one call, lookup_callsign(12,n12)→other; flip by iflip; CQ if icq;
  RRR/RR73/73 from nrpt.
  `n12=(p0<<4)|(p1>>4)`; `n58=((p1&0xF)<<54)|(p2<<46)|...|(p7<<6)|(p8>>2)` (all
  u64); `iflip=(p8>>1)&1`; `nrpt=((p8&1)<<1)|(p9>>7)`; `icq=(p9>>6)&1`.
- decode_free (port message.c:580): `decode_telemetry`→b71[9]; 13×{ rem(u16)=0;
  for i in 0..9 { rem=(rem<<8)|b71[i]; b71[i]=rem/42; rem%=42 } c14[idx]=charn(rem,
  Full) } building right-to-left; trim.
- `decode_telemetry(p)->[u8;9]`: carry=0; for i 0..9 { t[i]=(carry<<7)|(p[i]>>1);
  carry=p[i]&1 }. telemetry_hex = hex of those 9 bytes (18 chars).
- `unpack58(n58)`: 11 chars base38 (AlphanumSpaceSlash) right-aligned, trim,
  save_callsign.
- `lookup_callsign(type, hash)`: CallHash lookup; not found → `"<...>"`, found →
  `"<CALL>"` (add brackets).

**Encode (for the self-test round-trip): `encode_std`, `encode_free` (+helpers
pack28, pack_basecall, packgrid, parse_cq_modifier, encode_telemetry).** Port
message.c:192/335/378/720/779/811/1049. Payload packing for std (message.c:241):
```
n29a=(n28a<<1)|ipa; n29b=(n28b<<1)|ipb; (set n29a|=1 if call_to ends /R or /P)
p0=n29a>>21; p1=n29a>>13; p2=n29a>>5; p3=(n29a<<3)|(n29b>>26);
```

## Then `src/waterfall.rs` (port monitor.c)

`Monitor { nfft, block_size, subblock_size, min_bin, max_bin, num_bins, time_osr,
freq_osr, window: Vec<f32>, last_frame: Vec<f32>, fft: Fft, wf:

windows, FFTs (use `fft.forward_real`), writes scaled-u8 mags. Layout exactly:
`mag[block][time_sub][freq_sub][bin]` flattened, `block_stride =
time_osr*freq_osr*num_bins`. `num_blocks` grows; max_blocks =
slot_time/symbol_period (FT8 ≈ 93).

## Then `src/decode.rs` (port decode.c + decode_ft8.c::decode)

- `Candidate { score:i16, time_offset:i16, freq_offset:i16, time_sub:u8,
  freq_sub:u8 }`.
- `find_candidates(wf) -> Vec<Candidate>`: ft8_sync_score (the neighbor-diff
  variant in decode.c), heap top-140, sort desc. Ranges in "Key decisions".
- `extract_llr(wf, cand) -> [f32;174]`: ft8_extract_likelihood + ft8_extract_symbol
  (the `#if 1` max4 form). Skip Costas: `sym_idx = k + (k<29 ? 7 : 14)`,
  `bit_idx=3k`, k in 0..58. Then normalize_logl.
- `decode_candidate -> Option<payload>`: bp_decode → if errors>0 fail; pack 91
  bits; extract+recompute CRC, compare; (FT4: XOR payload back). hash = CRC.
- top-level `decode(samples@12k, protocol) -> Vec<Decode>`: monitor over all
  blocks → find_candidates → for each: decode_candidate, **dedup by payload hash**
  (mirror decode_ft8.c hashtable), unpack message. Compute `freq_hz = (min_bin +
  freq_offset + freq_sub/freq_osr) / symbol_period` and `dt = (time_offset +
  time_sub/time_osr) * symbol_period`. SNR ≈ `score*0.5` (POC; refine later).
- **Public `Decode { snr_db, dt, freq_hz, message: String, msg_type }`** (serde
  Serialize). Add helpers: `is_cq()`, parse call_to/call_de/grid from text.
